Pong Me for Windows
===================

Thanks for downloading Pong Me! Here's everything you need to know.


HOW TO INSTALL
--------------

1. Extract this entire folder somewhere permanent.
   We recommend creating a folder like:

       C:\Games\PongMe\

   Important: do NOT run PongMe.exe directly from inside the .zip file.
   Extract the whole folder first, or the game will not run.

2. (Optional) Make a Desktop shortcut:
   - Right-click PongMe.exe
   - Choose "Send to" -> "Desktop (create shortcut)"
   - You can now launch Pong Me from your desktop.

3. (Optional) Pin to Start Menu:
   - Right-click PongMe.exe
   - Choose "Pin to Start"


HOW TO PLAY
-----------

Double-click PongMe.exe to launch.

  The first time you run it, Windows SmartScreen may show a blue
  warning saying "Windows protected your PC". This is because the
  game is not yet code-signed (we're a tiny indie outfit!). To run:

      1. Click "More info"
      2. Click "Run anyway"

  You only need to do this once. The game is safe -- the source
  code is open and you can build it yourself if you'd like.


CONTROLS
--------

  W / S         Move left paddle up / down
  1 / 2 / 3     Ball speed: Slow / Medium / Fast
  M             Toggle sound on/off
  R             Reset / new game
  Alt+F4        Quit

The right paddle is AI-controlled.


HOW TO UNINSTALL
----------------

Just delete the folder you extracted. That's it. Pong Me does not
install anything to your registry or system folders.


WHAT'S IN THIS FOLDER
---------------------

  PongMe.exe              The game launcher (this is what you run)
  PongMe_Data/            Game assets, scenes, and scripts
  UnityPlayer.dll         Unity engine runtime
  UnityCrashHandler64.exe Crash reporter (optional)
  MonoBleedingEdge/       Mono runtime
  D3D12/                  Direct3D 12 graphics support
  dstorage.dll            DirectStorage runtime
  README.txt              This file

All of these files must stay together in the same folder for the
game to run. If you move or delete any of them, PongMe.exe won't
start.
